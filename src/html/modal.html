Group:

The periodic table can be divided into groups, or columns.

Period:

The periodic table can be divided into periods, or rows.

Atomic Number:

An element's atomic number is the number of protons an atom of that element has.

Atomic Mass:

The atomic mass (or relative atomic mass) of an element is the weighted mean mass of an atom of that element compared with 1/12 the mass of an atom of Carbon-12.

Block:

A block is a set of groups wherein the highest-energy electrons in each element belong to the same orbital type (the 's', 'd, 'p' and 'f' orbitals).

Type:

Elements can also be grouped by type. Elements of the same type are often in the same or a similar group, and have very similar chemical properties.

State at room temperature:

Elements all have different melting and boiling points, so not all are the same state at room temperature.

Electronegativity:

The electronegativity of an atom is its ability to attract the shared pair of electrons in a covalent bond. Here it is measure using the Pauling scale.
